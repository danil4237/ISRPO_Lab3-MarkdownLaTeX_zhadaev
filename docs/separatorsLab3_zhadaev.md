Markdown — это простой и удобный язык разметки, предназначенный для создания структурированных текстовых документов с минимальным использованием форматирования. Он был создан в 2004 году Джоном Грубером с целью позволить людям «писать с помощью обычного текста, легко читаемого без предварительной обработки». 
***
Основные возможности Markdown — это набор простых текстовых конструкций, которые позволяют легко форматировать документ без использования сложных тегов (например, HTML)
