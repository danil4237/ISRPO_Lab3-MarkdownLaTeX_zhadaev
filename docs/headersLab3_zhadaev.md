## Лабораторная работа №3. Работа с Markdown-разметкой и базовое использование LaTeX в документации проекта
#### Markdown (маркдаун) — облегчённый язык разметки для форматирования текста. В отличие от сложных языков разметки, таких как HTML или XML, Markdown использует простые символы для создания структурированных документов.


#### Язык разметки Markdown применяется в разных сферах, где нужно форматировать текст, создавать документацию, работать с текстом в мессенджерах или системах контроля версий.

#### Markdown нужен разработчикам программного обеспечения для удобства работы, создания документации и интеграции с инструментами для разработки. Язык разметки позволяет форматировать текст, делая его читаемым даже в исходном виде, и подходит для создания структурированных документов.
